### React Redux Webpack2 Hot Reloader

A boilerplate for personal SPA applications:

 - React 
 - Redux 
 - Webpack2
 - Hot Reloader
 - Code Splitting
 - Eslint
 - React Router (Not setup yet)
 - Express (Not setup yet)
 - Yarn

# Commands
To Install
 - yarn install


For Development
 - npm start


For Production
 - npm build
